作为前端开发者，选择合适的工具可以大大提高开发效率。这里推荐一些我常用的工具。

## 编辑器

### VS Code
微软出品的免费编辑器，插件生态丰富，性能优秀。

**推荐插件：**
- ESLint - 代码检查
- Prettier - 代码格式化
- Auto Rename Tag - 自动重命名标签
- Live Server - 本地开发服务器

<!--more-->

## 包管理器

### pnpm
比 npm 和 yarn 更快，更节省磁盘空间。

```bash
npm install -g pnpm
pnpm install
```

## 构建工具

### Vite
下一代前端构建工具，开发体验极佳。

```bash
pnpm create vite my-app
```

### Webpack
老牌构建工具，配置灵活强大。

## CSS 工具

### Tailwind CSS
实用优先的 CSS 框架。

### PostCSS
CSS 转换工具。

## 调试工具

### Chrome DevTools
浏览器自带的强大调试工具。

### Vue DevTools / React DevTools
框架专用调试工具。

## 版本控制

### Git
必备的版本控制系统。

### GitHub Desktop
Git 的图形化界面，简化操作。

## 其他工具

- **Figma** - UI 设计工具
- **Postman** - API 测试工具
- **Lighthouse** - 性能分析工具

选择适合自己的工具，提升开发效率！
